Summary: Latitude vs. X

The Purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish the analysis, data from OpenWeatherMap API was pulled to assemble an initial dataset of over 500 cities.

After creating the dataset, Matplotlib was utilized to create various plots to compare latitude vs. weather. Factors observed included temperature, cloudiness, windspeed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and decriptions of any trends and correlations witnessed.
